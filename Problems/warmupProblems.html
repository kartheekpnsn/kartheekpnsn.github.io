Problem-1
The parameter weekday is True if it is a weekday, and the parameter vacation is True if we are on vacation.
We sleep in if it is not a weekday or we're on vacation. Return True if we sleep in.

sleep_in(False, False) should return True
sleep_in(True, False) should return False
sleep_in(False, True) should return True


Problem-2
We have two monkeys, a and b, and the parameters a_smile and b_smile indicate if each is smiling.
We are in trouble if they are both smiling or if neither of them is smiling. Return True if we are in trouble.

monkey_trouble(True, True) should return True
monkey_trouble(False, False) should return True
monkey_trouble(True, False) should return False


Problem-3
Given two int values, return their sum. Unless the two values are the same, then return double their sum.

sum_double(1, 2) should return 3
sum_double(3, 2) should return 5
sum_double(2, 2) should return 8


Problem-4
Given an int n, return the absolute difference between n and 21, except return double the absolute difference if n is over 21.

diff21(19) should return 2
diff21(10) should return 11
diff21(21) should return 0


Problem-5
We have a loud talking parrot. The "hour" parameter is the current hour time in the range 0..23.
We are in trouble if the parrot is talking and the hour is before 7 or after 20. Return True if we are in trouble.

parrot_trouble(True, 6) should return True
parrot_trouble(True, 7) should return False
parrot_trouble(False, 6) should return False


Problem-6
Given a string, return a new string where "not " has been added to the front. However, if the string already begins with "not", return the string unchanged.

not_string('candy') should return 'not candy'
not_string('x') should return 'not x'
not_string('not bad') should return 'not bad'

front_back('code') should return 'eodc'
